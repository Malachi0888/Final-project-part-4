Why this project fits the assignment

This website uses semantic page structure, responsive design, custom styling, Bootstrap components, JavaScript interactivity, and external API data. It is built to be easy to deploy on GitHub Pages and simple to understand in a portfolio or classroom setting.

  • Three separate HTML pages
  • Custom CSS with media queries
  • Bootstrap navigation and cards
  • External weather API integration
  • Accessibility-minded labels and alt text
