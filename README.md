# Student Success Hub

## Project Description
Student Success Hub is a multi-page responsive website designed to help students stay organized, focused, and supported throughout the school week. The site includes productivity tools, wellness resources, and live weather data for a chosen city. It was built as a final front-end project to demonstrate responsive web design, accessibility, JavaScript interactivity, and API integration.

## Purpose
The purpose of this project is to create a polished, user-friendly web application that works well on desktop, tablet, and mobile devices while also showing strong front-end development practices.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Open-Meteo API
- GitHub Pages

## Pages and Features
### Home Page
- Introductory hero section
- Project overview
- Semantic layout and accessible navigation

### Tools Page
- Focus timer using JavaScript
- Goal tracker using localStorage
- Live weather search using the Open-Meteo API

### Resources Page
- Resource cards
- Bootstrap FAQ accordion
- Contact form demo with JavaScript validation

## Accessibility Features
- Semantic headings and landmarks
- Skip link
- Alt text on images
- Clear labels on forms
- Strong color contrast

## Setup Instructions
1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Launch `index.html` in a browser or use the Live Server extension.
4. To deploy, push the project to GitHub and enable GitHub Pages in the repository settings.

## Deployment
This project is designed to be deployed on GitHub Pages because it uses static front-end files only.
